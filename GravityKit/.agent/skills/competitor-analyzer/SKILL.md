# Competitor Analyzer

## Purpose
Research and analyze competitor products to identify standard features, gap analysis, and unique selling points (USP).

## Usage

### 1. Analyze Competitors
```bash
python .agent/skills/competitor-analyzer/scripts/analyzer.py --keywords "e-commerce, fashion" --limit 3
```

### Output
- List of top competitors.
- Key Features table.
- Strengths & Weaknesses analysis.
- Market Gaps.
