# UI/UX Pro Max

Comprehensive design guide.

## Usage

### 1. Generate Design System
```bash
python .agent/skills/ui-ux-pro-max/scripts/search.py "<product_type> <keywords>" --design-system -p "<Project Name>"
```

### 2. Detailed Search
```bash
python .agent/skills/ui-ux-pro-max/scripts/search.py "<keyword>" --domain <domain>
```

### Domains
- `style`: Visual styles
- `typography`: Fonts
- `color`: Palettes
- `component`: UI components
- `ux`: Best practices

### Outputs
ASCII tables or Markdown.
